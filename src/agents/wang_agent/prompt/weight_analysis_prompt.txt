设计提示：{design_prompt}

请仔细查看图像，分析其视觉结构，将其分解为按权重排序的组件。权重越高的组件越重要，应该优先生成。

对于每个组件，请提供：
- weight: 权重值（0-1之间，越高越重要）
- description: 该组件的详细描述，包括：
  * 组件在图像中的位置（大致位置，如"中心"、"左上角"等）
  * 组件的主要形状特征（圆形、矩形、曲线、复杂路径等）
  * 组件的主要颜色
  * 组件的关键视觉特征（如"字母A"、"圆形背景"、"装饰线条"等）

请确保：
1. 组件按权重从高到低排序
2. 每个组件都有清晰的描述，包含位置、形状、颜色等关键信息
3. 组件之间不重叠或重叠最小
4. 总权重合理分布
5. 至少提供1个组件，最多5个组件
6. 描述要足够详细，以便后续从PNG中提取该组件的结构信息（关键点、轮廓等）

**重要**：请以JSON格式返回，格式如下：

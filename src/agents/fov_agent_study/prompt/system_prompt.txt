# 【专业设计指导】

## “正负形”设计理念
画面中的正形一般是第一眼看到的形状，而背景/空白形成的空间则是第二眼看出的形状。画面中的形状可以抽象或者复杂，**边界是其灵魂**。正负形需要有和谐而一致的“共享边界”，引导人视角的平衡。画面可以使用全部背景作为“负空间”，也可以使用LOGO中常用的几何形状的背景，然后雕刻出“负空间”。
**重要**: 大部分时候需要一定的*变形*和*抽象*化使得正负形的共享边界更加和谐，且保留物体的语义完整度。

## “正负形”设计三大核心技巧

**1. 叠加共构类 (Shared Form)**
- **定义**: 两个或多个图形元素互相重叠，重叠的部分成为它们共同的构成元素，形成一个不可分割的整体。

**2. 同形互借类 (Shared Contour)**
- **定义**: 两个图形共享一条或多条轮廓线，一个图形的结束就是另一个图形的开始。它们互相借用对方的边界来定义自己。

**3. 阴阳互换类 (Figure-ground Reversal)**
- **定义**: 正形与负形处于完全平等且可相互转换的地位，整个画面形成一种动态的平衡，无主次之分。
